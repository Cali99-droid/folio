Últimos Proyectos

DIGITDBU

Aplicación que integra su base de datos con la plataforma Power BI para presentar un análisis de estos a través de un informe que se muestra en la misma aplicacion web, ademas cuenta con una puerta de enlace con actualizacion programada y automatica.

Es un proyecto elaborado para la digitalizacion de fichas socioeconómicas de la Dirección de Bienestar Universitario de la Universidad Nacional Santiago Antunez de Mayolo.

  • Laravel 9
  • Power BI
  • MySQL
  • Tailwind
  • Javascript

Organizaciones UNASAM

Proyecto destinado a la gestión de organizaciones estudiantiles de la Universidad Nacional Santiago Antúnez de Mayolo, cuenta con una amplia cantidad de módulos y una interfaz amigable.
Hace uso de Fetch API para consumir recursos de forma asincrona, creación y delegación de roles de usuarios, dashboard, almacenamiento de archivos y landing page.

  • PHP
  • MySQL
  • Sass
  • Javascript

USTEC OGTISE

Aplicación para la gestión de incidentes, hecha para la unidad de soporte técnico de la oficina de tecnologías de la información de la Universidad Nacional Santiago Antúnez de Mayolo.
Sus principales caracteristicas son: inventario e historial de incidentes en los equipos de cómputo, redes y/o software, detección de mantenimiento preventivo, notificación via email de incidentes, gestion de stock de materiales para el mantenimiento en la atención de incidentes.

  • Laravel 9
  • MySQL
  • Tailwind
  • Javascript

Almacén

Proyecto desarrollado para la gestion de almacen para el Consorcio Queralcoto, cuenta con manejo de usuarios y roles, control de stock, gestion de entradas y salidas e informe gráfico.

  • PHP
  • MySQL
  • Bootstrap
  • Javascript
